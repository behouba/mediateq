# mediateq

mediateq is a file storage REST API micro-service.

## Installation

### Install libvips

Clone the project

```bash
git clone https://github.com/behouba/mediateq.git
```

### Testing

### build mediateq

To build mediateq you can run the build.sh script

```bash
./build.sh
```

### Run mediateq

```bash
./bin/mediateq -config=mediateq.yaml
```
